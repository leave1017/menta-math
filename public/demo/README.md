# Demo Video Directory

## Required Files

Please place your demo video file in this directory:

### trainer-demo.mp4
- **File name**: `trainer-demo.mp4`
- **Max size**: < 900KB
- **Format**: MP4 (H.264)
- **Recommended specs**:
  - Resolution: 1920x1080 or 1280x720
  - Duration: ~15 seconds
  - Bitrate: ~400-500 Kbps (to keep under 900KB)

### trainer-demo-poster.jpg (Optional)
- **File name**: `trainer-demo-poster.jpg`
- **Purpose**: Poster image shown before video plays
- **Recommended specs**:
  - Same aspect ratio as video (16:9)
  - Resolution: 1280x720
  - Format: JPEG
  - Size: < 100KB

## URL Access

Once files are placed here, they will be accessible at:
- Video: `/demo/trainer-demo.mp4`
- Poster: `/demo/trainer-demo-poster.jpg`

## Note

The video module has been added to the homepage between the Trainer and SEO content sections. It will display once you add the video file.
